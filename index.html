HTML é a linguagem de marcação padrão para a criação de páginas da Web.

Ele consiste em três partes:

  • tag html: tudo que tiver dentro dessa tag é html.
  • tag head que é o cabeçalho, tudo que é carregado antes de aparecer para o usuário
  • tag body: tudo que tiver entre o body são coisas que vão aparecer para o usuário.

Tags - [voltar]

São elementos usados para estruturar e organizar o conteúdo de uma página web. Funcionam como marcadores, indicando ao navegador o que é um título, um parágrafo, uma imagem, um link, etc. Algumas tags não possuem fechamento("/>") - (img - br> - hr> - input>) - são chamadas de tags vazias ou self-closing.

Atributos das Tags

As tags possuem atributos que são propriedades/informações que uma tag pode ter – temos as globais e específicas de cada tag. Estrutura do atributo:<tag atributo = “valor”> - ex: <strong id= “titulo”> Meu HTML /strong>

Globais: id; class; style; title; hidden; data-*

Específicas: href; src; alt; type; placeholder; disabled

Tipografia - textos - [voltar]

Conjunto de elementos que definem a apresentação do texto, como tipo de fonte, tamanho, estilo (itálico), espaçamento e alinhamento. No HTML, a tipografia é definida principalmente por tags semânticas de texto e CSS (estilização)

Títulos: h1 até h6 - h1:título e h2:subtítulo

Parágrafo:<p> - define os blocos de texto

Ênfase e importância:<em> - ênfase e <strong> - negrito

Listas - [voltar]

Elementos usados para organizar informações em sequência ou agrupamentos. Servem para organizar conteúdo, melhorar legibilidade, estruturar menus, criar navegação e exibir etapas ou itens. Existem a lista Ordenada, a Não Ordenada e a de Definição.

Lista Ordenada (ol): Quando a ordem importa. Usado para passo a passo, ranking, sequência.

Atributos importantes: type define o tipo de marcador - <ol type="A">/ start define número inicial - <ol start="5">

Lista Não Ordenada (ul): Usado quando a ordem não importa. Usado para agrupamento simples.

Links - [voltar]

Elementos que permitem navegar entre páginas, ir para outros sites, ir para arquivos, ir para partes específicas da mesma página, enviar e-mail, iniciar ligações. São criados com a tag <a>.

<a href="https://google.com">Ir para o Google</a>

Atributos Importantes

→ href (obrigatório) – define o destino do link que pode ser externo e interno.

Externo:<a href= "https://www.google.com">Google</a>

Interno:<a href="contato.html>Contato</a>
